Project name:Internal web.

Wordress | Elementor | BetterDocs

Internal web for the team that I work in.

We use internal web for storing useful information and links, archiving questions, contacts, mailing list administration etc. This page lacks certain features most notably a search option which would provide huge help in searching for a right answer in our knowledge base. While the new internal web is buing built, I suggested we transfer our "part" of the internal web to another site.

We used Wordpress beacuse I had some experience with it, plus it was used for some other projects so we got it set up quickly. I was feeling boxed in by the theme provided so I installed Elementor and built everything from the bottom up. Most of my time was spent building a knowledge base with a search option.

Eventually we tranfered completely and continue to use it to this day. I even got one shift a week where I take care of updates, implement new features, sort out bugs and make sure everything works properly. We also have a place where everyone can make a suggestion and report a bug.

Although web design is not my interest, I got a chance to work with knowledge base and designed it to fit our basic needs like having a good search option and a clean, modern look. No source code is provided for this project because I did't actually code it. All my work was done using plugins and built-in CMS options. Still, I am proud of being proactive and "pushing" the project enough so it gets done.
